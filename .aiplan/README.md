# AI Plan Workspace

This folder is the temporary execution workspace for multi-agent planning, argument, and consensus.

## Active Work Files

- `plan.md`: current task plan, owners, dependencies, and status
- `consensus.md`: the final agreed direction before or during execution
- `argue/<role>.md`: short decision notes from specialists when viewpoints differ

## Lifecycle

1. Create or refresh the plan files while the task is active.
2. Keep notes short and focused on decisions.
3. Clean up temporary files when the task is fully executed.

Keep this file and `argue/README.md` as the scaffold. Remove the rest after completion.
